Privacy Policy for Signal

Last Revision - 1st January 2021
The policy can be updated at any time without notification.

By using Signal, you agree to the policy set out here.

What Do We Store

The signal developers only store ID's handed out by discord to manage prefixes, automod, logging and punishments.
This is limited to Server ID's, and User ID's.

The Server ID is used for prefixes, automoderator and statistics.
The User ID is used for punishment logging, along with certain statistics.

Who can Access the Data Stored

Only the bot developers (PenPow) has access to all information.
Server Moderators/Administrators can also access information todo with moderation cases for the guilds they moderate in.

How to Opt Out of Data Collection

Should you wish to opt out of non essential data collection (statistics and developer logging), contact PenPow#7067 on discord. Data for moderation cases and prefixes cannot be removed.

Why do we Store this Info

The developers at Signal store this data since without it the bot loses most functionality.
The ID's are used to save settings set by the guild, and provide core functionality to the moderation section.

Questions about Data Collection

Should you have any questions about the data we store, or questions about removing or obtaining your data, message PenPow#7067 on discord for more information.
